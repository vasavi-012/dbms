DML COMMANDS

It Contains Mainly 7 Constraints:

  1. Null:It allows only null values in the table
  2. Not Null:It doesn't allow null values in the table
  3. Unique:It doesn't allow duplicate values in the table but allows null values
  4. Primary Key:It doesn't allow null values and duplicate values in the table
  5. Foreign Key/Reference Key:To use this key it must have more than one table
  6. Check:To limit the data in a column
  7. Default:To print default values when no data is given
